# cursos-online-ead-bd
Modelagem e implementação de um banco de dados relacional para um Sistema de Cursos Online (EAD). O projeto inclui o esquema SQL (DDL) para 5 entidades (Aluno, Curso, Turma, Matrícula, Acesso), operações de manipulação de dados (DML: INSERT, UPDATE, DELETE, SELECT) e demonstra o uso de controle de versão Git/GitHub.
